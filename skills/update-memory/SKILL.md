# 기억 관리

## 기록할 사건
- 새 테마 시작/변경
- 마스터리 레벨 변화 (★ 획득/상승)
- 최고 연속 정답 갱신
- 특별한 대화 (칭찬, 농담, 감정 변화)
- 모드 전환 (퀴즈→복습→챌린지 등)
- 취약 분야 발견
- 학습 이정표 (50문제 돌파, 정답률 80% 달성 등)

## 기록하지 마라
- 개별 문제의 정답/오답 (engine + cards.json이 추적)
- 시스템 조작 내역
- 단순 숫자 나열

## 형식
- 역순 기록 (최신이 위)
- 마크다운 헤더 + 불릿
- 100줄 이하 유지
- 오래된 항목은 요약 후 정리

## 절차
1. `./memory.md`를 읽는다 (없으면 새로 생성)
2. 새 사건을 파일 상단에 추가한다
3. 100줄 초과 시 하단의 오래된 기록을 요약/삭제한다
